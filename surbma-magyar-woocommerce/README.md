HuCommerce — Hungarian WooCommerce Extensions
===============================================

Useful fixes and enhancements for Hungarian WooCommerce webshops.

This plugin solves common pain points specific to Hungarian e-commerce:
name order (last name first), tax number field on checkout, ZIP-based city
autocomplete, checkout field validation, legal compliance (GDPR/consumer
protection), and much more.

- **Free version (HuCommerce Start):** Core fixes for every Hungarian WooCommerce store.
- **Premium version (HuCommerce Pro):** Extended feature set including legal compliance tools, UI customisations, SMTP, and priority support.

More information: [hucommerce.hu](https://www.hucommerce.hu/)
